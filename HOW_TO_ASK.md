# Claudeへの要望の出し方

## 窓口は3つ

| 窓口 | 使う場面 | やり方 |
|---|---|---|
| **`natsu req`** | PCで思いついた一言 | `natsu req "花火をもっと長く"` |
| **`natsu talk`** | PCでじっくり書きたい | メモ帳が開くので書いて保存 |
| **GitHubの `REQUESTS.md`** | スマホから | アプリで開いて鉛筆アイコン → 編集 → Commit |

どれも同じ `REQUESTS.md` に書き込まれます。
次の自動実行でClaudeが最優先で読み、**同じファイルに返事を書きます。**
返事はGitHubからスマホでも読めます。往復の会話になります。

---

## 書き方のコツ

### 基本: 「何を」より「どう感じたか」を書く

Claudeはコードは書けますが、あなたが遊んだときの体感は分かりません。
一番役に立つのは**感想**です。

```
○ 良い例
金魚すくいのシーン、あっさりしすぎててもったいない感じがした

△ もったいない例
金魚すくいを改善して
```

前者なら「どう物足りないのか」から具体案を出せます。

### 場面を特定できると精度が上がる

```
○ 「迷子イベントのあと、茜と合流する場面が唐突に感じた」
△ 「話の繋がりが変」
```

### 好き嫌いも遠慮なく書いていい

```
○ 「エンディング3の終わり方、切なすぎて好みじゃない。もっと前向きな方がいい」
○ 「BGMの音、ちょっとうるさく感じる。もう少し控えめに」
```

Claudeが「これは好みの問題だから勝手に決めない方がいい」と判断した項目は
`IMPROVEMENTS_LOG.md` の「要ユーザー判断」に溜まっていきます。
そこにA案B案が書かれているので、選んで `natsu req` で返せばOKです。

### 曖昧なままでも投げていい

```
○ 「なんか全体的に物足りない。何が足りないと思う?」
○ 「もっと泣ける方向にしたいけど、どうすればいいか分からない」
```

Claudeが分析して案を出し、`REQUESTS.md` に返事を書きます。
方向性の相談も窓口として使えます。

### 逆に、Claudeから質問が来ることもあります

```
> Claude 07/22 03:10 — 金魚すくいの反応を厚くしました。
> ついでに「金魚を逃がしてしまう」分岐も作れそうですが、
> 切ない方向になるので追加するか迷っています。欲しければ一言ください。
```

これに答えると会話が進みます。放置しても勝手に止まったりはしません。

## イラストの要望

```
natsu req "告白シーンに一枚絵が欲しい"
```

Claudeが `ART_QUEUE.md` にプロンプトを書き出します。
そこから先は `MANUAL_SETUP.md` の「イラストの自動化」の通り、
自動生成(ルートA)か手動(ルートB)で画像になります。

**絵の雰囲気に注文があるときは一緒に書いてください:**

```
natsu req "告白シーンの一枚絵。夜の神社の階段で、
茜が振り返ってるところ。切ない感じより、まっすぐな感じで"
```

これがそのままプロンプトに反映されます。

**出来上がった絵が気に入らないとき:**

```
natsu req "hanabi_kiss の絵、表情が硬い。もっと柔らかい笑顔で作り直して"
```

キー名を書くと確実に伝わります(`natsu arts` で確認できます)。

---

## 優先順位

Claudeは毎回の実行でこの順に処理します。

1. `ART_NEW.md` — 届いた画像の組み込み
2. `REQUESTS.md` の `(未読)` — **あなたの要望**
3. `BACKLOG.md` — 溜めてある改善案
4. どれも無ければ、自分で改善点を探す

つまり **`natsu req` で投げたものが最優先で処理されます。**

急ぎのときは投げたあとに `natsu now` を打てば、
次の定期実行を待たずすぐ処理されます。

---

## 溜めておくだけの案

「いつかやってほしいけど急がない」ものは `BACKLOG.md` に書きます。

- PCから: `notepad BACKLOG.md`
- スマホから: GitHubで `BACKLOG.md` を編集

1行ずつ `- [ ] やってほしいこと` の形で書けば、
要望が無いときに上から順に処理されます。

---

## 困ったときの相談先

仕組み自体が動かない・エラーが出るといった話は、
Claude Codeではなく**このチャット(Claudeアプリ)**に聞いてください。

```
natsu doctor
```

の結果をスクショで送ってもらえれば、だいたい原因が分かります。
